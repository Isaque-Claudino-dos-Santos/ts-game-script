> # Command to game
>
> ```
> npm run game [verb]=[action] [options [flag=value]]
> ```

> ## Options
>
> | Flag | Values                 | Required | Default |
> | ---- | ---------------------- | -------- | ------- |
> | name | name of class and file | True     | Error   |
> | draw | rect,text,arc,image    | False    | rect    |

> ## Create
>
> ```
> npm run game create=[optionCreate] [options [flag=value]]
> ```
>
> ### What Write
>
> | create=... | options    |
> | ---------- | ---------- |
> | object     | name, draw |
> | room       | name       |
